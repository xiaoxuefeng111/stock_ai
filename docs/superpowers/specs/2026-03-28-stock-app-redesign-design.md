# 股票分析应用 UI重构与功能扩展设计文档

**日期**: 2026-03-28
**版本**: 1.0

---

## 一、项目概述

### 1.1 目标
将现有单页面股票分析应用重构为多Tab架构，优化UI布局，新增技术指标、策略分析、AI深度分析等功能。

### 1.2 核心改动
- Tab布局替代单页面堆叠
- 自选股页面增强（大盘指数+市场情绪）
- 技术指标专业化（参数可调+策略组合+买卖点标记）
- AI深度分析独立Tab（多维度分析+实时问答）
- 支持深色/浅色主题切换

---

## 二、整体架构

### 2.1 Tab结构

```
[自选股] [个股分析] [技术指标] [AI深度分析]
```

| Tab | 定位 | 内容 |
|-----|------|------|
| 自选股 | 首页/快速浏览 | 大盘指数 + 市场情绪 + 自选股卡片列表 |
| 个股分析 | 快速概览 | 行情概览 + 资讯时间线 + 简洁图表 + 基础指标 |
| 技术指标 | 专业深度 | 参数可调指标 + 组合策略 + 买卖点标记 |
| AI深度分析 | 智能决策 | 多维度分析 + 行业对比 + 历史相似 + 实时问答 |

### 2.2 全局组件
- **顶部标题栏**: 应用名称 + 主题切换按钮（☀️/🌙）
- **底部状态栏**: 数据来源 + 最后更新时间 + 免责声明

```

├─────────────────────────────────────────────────────────┤
│  🌡️ 市场情绪                                            │
│  上涨 2,856家  下跌 2,134家  涨停 45  跌停 23          │
│  市场热度: ████████░░ 较热 (75分)                       │
├─────────────────────────────────────────────────────────┤
│  [🔄 刷新] [➕ 添加] [筛选: 全部/涨幅/跌幅]             │
├─────────────────────────────────────────────────────────┤
│  自选股卡片列表...                                      │
└─────────────────────────────────────────────────────────┘
```

#### 3.1.2 大盘指数
- 数据源: AkShare `stock_zh_index_spot_em`
- 显示内容: 上证指数、深证成指、创业板指、科创50
- 实时更新: 30秒自动刷新

#### 3.1.3 市场情绪仪表盘

**原始数据来源:**
- 涨跌家数: `stock_zh_a_spot_em` 统计
- 涨停/跌停: `stock_zt_pool_em`
- 北向资金: `stock_hsgt_north_net_flow_in_em`
- 主力资金: `stock_individual_fund_flow`

**量化情绪指标（计算）:**
| 指标 | 计算方式 |
|------|----------|
| 恐惧贪婪指数 | RSI + 波动率 + 动量综合 |
| 量价情绪 | 成交量变化率 vs 价格变化率 |
| 动量情绪 | 多周期涨跌幅加权 |
| 波动情绪 | ATR/标准差 |
| 筹码情绪 | 成本分布计算 |
| 资金强度 | 主力资金/总成交 |
| 连板情绪 | 涨停股统计 |
| 封板强度 | 封单/流通市值 |

**显示格式:**
```
🌡️ 市场情绪总览
恐惧贪婪指数: 65 (贪婪)  ▓▓▓▓▓▓░░░░
市场热度: 75分  上涨2856家 下跌2134家
涨停45家  跌停23家  连板18家  炸板率32%
北向净流入: +23.5亿  主力净流入: -12.3亿
```

#### 3.1.4 自选股卡片

**卡片布局:**
```

│ 华胜天成 600410 沪市主板  🔥主力流入  📈看多     │
│ 当量 230.9万手    30天高量 602.9万手            │

│ │ 26.60   │ -6.60%  │ 28.74   │                │
│ └─────────┴─────────┴─────────┘                │
│ ┌─────────┬─────────┬─────────┐                │
│ │  MA5    │  MA13   │  MA20   │                │
│ │ 26.78   │ 27.89   │ 29.57   │                │
│ └─────────┴─────────┴─────────┘                │

**个股情绪标签:**
```

**数据源:**

#### 3.3.2 技术指标清单

| 类别 | 指标 | 参数可调 |
|------|------|----------|
| 均线 | MA5, MA10, MA20, MA60 | 周期数 |
| 趋势 | MACD, BOLL, DMI, SAR | MACD快慢线、BOLL宽度等 |
| 动量 | RSI, KDJ, WR, CCI | 周期参数 |
| 成交量 | VOL, OBV | - |

#### 3.3.3 指标组合策略

**预设策略:**
- MACD金叉 + RSI超卖
- KDJ金叉 + 成交量放大
- BOLL下轨支撑 + MACD底背离
- MA均线多头排列

**策略条件示例:**
```
策略: MACD金叉+RSI超卖
条件:
• MACD金叉 (DIF上穿DEA)
• RSI < 30 (超卖区)
• 成交量放大 1.5倍
```

- 在K线图上用箭头标记买卖信号
- 显示信号触发日期
- 支持开启/关闭标记显示

- 支持切换模型

## 八、风险与限制

### 8.1 数据源风险
- AkShare依赖东方财富等第三方数据源
- 数据源变更可能导致接口失效
- 海外服务器可能无法访问

### 8.2 AI分析限制
- AI分析仅供参考，不构成投资建议
- 模型输出可能存在偏差
- 需要用户自行判断风险

### 8.3 性能考虑
- 全市场数据加载需1-2分钟
- 建议使用初始化按钮预加载
- 多用户并发可能影响性能
